Название проекта: Buffer Changer 


Описание проекта: 

Программа для временной подмены буфера обмена. 
Программа запоминает/сохраняет текущий текст сохранённый в буфере обмена,
содержит несколько хоткеев[ctrl+c - скопировать текст\сохранить в базу програмы, 

						   ctrl+b - замена текста на рандомный символы 
						   (Важно чтобы длина изменнёного текста была равна оригиналу),

						   ctrl+m - вернуть исходный текст в буфер обмена].

Пример работы программы находится в одноименном видео в папке.


Требования к проекту: 

1. Управление программой через горячие клавиши
(Приветствуется использование библиотеки - keyboard или pyautogui).

2. Управление буфером обмена осуществляется через библиотеку - pyperclip

3. Python файл должен иметь расширение .pyw
(Расширение .pyw, представляет собой запуск программы в тихом режиме без консольного вывода).

4. Запуск\Закрытие программы сопровождается звуком.
(Приветствуется использование библиотеки - playsound 1.2.2)


Технические детали:

1. Программа будет разработана с использованием выбранного языка программирования Python (Любой версии).
2. Для доступа к буфера обмена будет использоваться Pyperclip.
3. Для хранения текущего текста можно использовать любую базу данных, можно использова dataclasses.


План работ:

1. Изучение документации к библиотеке Pyperclip.
2. Изучение документации к библиотеке PlaySound 1.2.2.
3. Создание основной структуры программы.
4. Реализовать систему хранение текста буфера обмена.
5. Программа должна быть максимально оптимизирована.


В процессе разработки можно задавать вопросы.

Техническое Собеседование: Ping-Бот для Мониторинга Серверов

Описание проекта:

Разработка бота для мониторинга серверов с использованием технологии пинга. 
Бот будет проверять работоспособность серверов в соответствии с предварительно заданным ресурсов.
Основная цель - предоставление оперативной информации о состоянии серверов.

Требования к проекту:

1. Программа будет разработана с использованием выбранного языка программирования Python (Любой версии).

2. Программа использует любую библиотеку для реализации телеграмм-бота
(Приветствуется использование библиотеки - aiogram 2.22.1).

3. Ведение подробных логов о результатах проверок.(задержка,время приёма передачи, пакеты, время ответа)

4. Простой интерфейс для указания сервера и мониторинга состояния серверов.

Интерфейс: (на выбор)

1.Консольный интерфейс: Возможность запуска и настройки бота из консоли.
2.Веб-интерфейс: Опциональный веб-интерфейс для управления и мониторинга через браузер.


План работ:

1. Создание основной структуры бота.
2. Программа должна быть максимально оптимизирована.


В процессе разработки можно задавать вопросы.
В текущей папке имеется пример кода, пингующий сервера

Вопросы кандидату:
1. Вопросы по коду
2. Вопросы от Марка Депозитов(haises)
